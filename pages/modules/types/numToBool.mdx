### numToBool
The numToBool submodule converts a number into a boolean.

**Examples**:

0:

```js
// Import util
const { util } = require("easyscriptjs");

// Convert 0 number to boolean
const result = util.numToBool(0);

// Log result to console
console.log(result);
// false
```

1:

```js
// Import util
const { util } = require("easyscriptjs");

// Convert 1 number to boolean
const result = util.numToBool(1);

// Log result to console
console.log(result);
// true
```
