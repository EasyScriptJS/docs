# boolToStr
The boolToStr submodule converts a boolean into a string.

## Examples

### true

```js
// Import util
const { util } = require("easyscriptjs");

// Convert true to number
const result = boolToStr(true);

// Log result to console
console.log(result);
// 'true'
```

### false

```js
// Import util
const { util } = require("easyscriptjs");

// Convert false to number
const result = boolToStr(false);

// Log result to console
console.log(result);
// 'false'
```
