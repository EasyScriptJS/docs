### boolToNum
The boolToNum submodule converts a boolean into a number.

**Examples**:

true:

```js
// Import util
const { util } = require("easyscriptjs");

// Convert true to number
const result = util.boolToNum(true);

// Log result to console
console.log(result);
// 1
```

false:

```js
// Import util
const { util } = require("easyscriptjs");

// Convert false to number
const result = util.boolToNum(false);

// Log result to console
console.log(result);
// 0
```
