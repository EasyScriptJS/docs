# strToBool
The strToBool submodule converts a string into a boolean.

## Examples

### "true"

```js
// Import util
const { util } = require("easyscriptjs");

// Convert "true" string to boolean
const result = util.strToBool("true");

// Log result to console
console.log(result);
// true
```

### "false"

```js
// Import util
const { util } = require("easyscriptjs");

// Convert "false" string to boolean
const result = util.strToBool("false");

// Log result to console
console.log(result);
// false
```
