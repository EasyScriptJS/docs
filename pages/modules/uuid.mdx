# uuid
The UUID module allows you to generate an ID using the format: `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx`.

## Example

```js
// Import module
const { uuid } = require("easyscriptjs");

// Generate ID
const result = uuid();

// Log result to console
console.log(result);
// a807a029-ff82-4fb1-b353-c1d91266dfce
```
